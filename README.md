🗳️ Tamil Nadu Election Results Analysis Dashboard 2026 | Power BI
📌 Project Overview

This project is an interactive Power BI dashboard that analyzes the Tamil Nadu Assembly Election Results 2026. The dashboard provides insights into party performance, constituency-wise winning margins, leading candidates, and overall election statistics to help users understand election trends through data visualization.

🎯 Objectives
Analyze party-wise seat distribution.
Identify the leading political party.
Compare winning margins across constituencies.
Display top winning candidates.
Provide interactive filtering for detailed analysis.

📊 Dashboard Features
✅ Total Constituencies
✅ Total Political Parties
✅ Average Winning Margin
✅ Highest Winning Margin
✅ Top 5 Winning Parties
✅ Party-wise Seat Distribution
✅ Top 10 Highest Winning Margins
✅ Constituency-wise Margin Analysis
✅ Candidate & Party Details Table
✅ Interactive Slicers and Filters
✅ Reset & Bookmark Buttons

🛠️ Tools & Technologies
Power BI Desktop
Microsoft Excel
Power Query
DAX (Data Analysis Expressions)

📈 Key Insights
Leading party based on seat count.
Constituencies with the highest winning margins.
Distribution of seats among major political parties.
Candidate performance across constituencies.
Interactive analysis using filters.
